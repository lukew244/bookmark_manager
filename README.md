# Bookmark Manager


## User Stories ##

```
As a User,
So that I can see what links I have saved,
I want to see a list of all my links.

As a User
So that I can save a website,
I want to be able to add new links with a title.

As a User,
So that I can categorise my links,
I want to create new tags.

As a User,
So that I can group links into different categories,
I want to be able to add tags to the link.

As a User,
So that I can find links on a particular topic,
I would like to filter links by tags.

As a User,
So that I can find recently bookmarked websites,
I would like to see links in descending date order with newest link first.

```
